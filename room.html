get /device/list

  • Request
  • Response

Query Parameters

  • pagination: (object)

    Information about pagination

    • page: required (number - minimum: 0)

      Current page to return

    • size: required (number - minimum: 1)

      Size of page, how many items each request will return

HTTP status code 200

Body

Media type: application/json

Type: array of info

Items: device

  • id: required (string)

    ID of this device

  • state: required (boolean)

    Current state of port

  • location: required (string)

    Location where this device is

    Example:

    bedroom
  • name: required (string)

    Name given for this port, this will be sync in all devices connected do PIServer

    Example:

    light
  • type: required (one of toggle, joint)

    Type of this port will be work

  • together: (array of object)

    If type of port is joint, other ports will be turn off before that is turn on

    Items: items

    • name: required (string)

      Name of port that is work together with this port

      Example:

      light
